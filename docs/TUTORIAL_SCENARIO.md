# 튜토리얼 시나리오

전체 8단계 + 인트로/완료. 총 예상 시간 **7~9분**.

## 단계 요약
| # | 단계 | 사용자 행동 | 성공 판정 | 예상 |
|---|---|---|---|---|
| 0 | 환영 / 둘러보기 | 머리를 돌려 방을 본다 | 누적 head yaw ≥ 90° | 30s |
| 1 | 컨트롤러 인식 | 양손을 시야에 든다 | 양 컨트롤러 트래킹 5초 유지 | 30s |
| 2 | 트리거 누르기 | 라이트빔 발사 | Trigger 입력 양손 각 1회 | 30s |
| 3 | 직접 그랩 | 책상 위 큐브를 박스에 넣기 | DropZone 진입 + Release | 60s |
| 4 | 레이 그랩 | 멀리 있는 큐브를 광선으로 끌어와 박스에 | 위와 동일 (Ray Interactor) | 60s |
| 5 | 텔레포트 | 빛나는 발판으로 이동 | TeleportTarget Trigger 진입 | 60s |
| 6 | UI 클릭 | 월드 패널의 "다음" 버튼 클릭 | Button.onClick | 30s |
| 7 | 통합 챌린지 | 큐브를 들고 텔레포트 → 슬롯에 끼우기 | XRSocketInteractor 매칭 | 90s |
| 8 | 완료 | "본 게임 시작" / "다시하기" 선택 | 버튼 클릭 → 씬 전환 | 30s |

각 단계는 **음성 내레이션 + 자막 + 시각 힌트(컨트롤러 글로우, 화살표 파티클)** 의 3중 가이드.
실패/미응답 5초 후 힌트 강조, 10초 후 자동 합격 처리한다.

---

## 단계별 상세

### 0. 환영 / 둘러보기
- **연출**: 어두운 방에서 페이드인. 정면에 환영 문구 부유.
- **내레이션**: "환영합니다. 천천히 고개를 돌려 주변을 둘러보세요."
- **시각 힌트**: 좌우 벽에 별 파티클이 점멸 → 사용자가 시선을 돌리도록 유도.
- **성공 조건**: 머리 yaw 회전 누적 ≥ 90° **또는** 8초 경과.
- **다음 단계로**: 정면 패널 페이드인 + 카운트다운 후 자동 전환.

### 1. 컨트롤러 인식
- **연출**: 양 손 위치에 컨트롤러 모델이 추가로 부유 → 사용자가 들어올리면 손 안에 흡수되는 연출.
- **내레이션**: "양손에 컨트롤러가 있습니다. 두 손을 눈앞으로 들어 올려보세요."
- **성공 조건**: `XRController.isTracked == true` 양손 모두 + 시야 범위 내 5초 유지.
- **실패 처리**: 한쪽만 트래킹되면 해당 손에 빨간 발광 표시.

### 2. 트리거 누르기
- **연출**: 양 컨트롤러 끝에 라이트건 모델 부착. 정면 1m 거리에 두 개의 표적.
- **내레이션**: "검지손가락 위치의 트리거를 당겨 표적을 맞춰보세요."
- **시각 힌트**: 컨트롤러 트리거 부분에 화살표 + 글로우.
- **성공 조건**: 양손 각각 1회 트리거 입력으로 표적 명중.

### 3. 직접 그랩 (Direct Grab)
- **연출**: 눈앞 책상 위에 색깔 큐브 3개 + 그 옆에 박스 1개.
- **내레이션**: "큐브 옆으로 손을 가져가 그립 버튼을 잡고, 박스 안에 놓아주세요."
- **시각 힌트**: 그립 버튼(가운데 손가락) 위치 글로우. 박스 위에 화살표.
- **성공 조건**: 큐브 1개 이상이 박스 DropZone 안에서 Release.
- **에러 케이스**: 큐브가 바닥으로 떨어지면 5초 후 책상 위로 자동 복귀.

### 4. 레이 그랩 (Ray Grab)
- **연출**: 손 닿지 않는 2m 거리 선반 위에 큐브 3개. 박스는 손 닿는 거리.
- **내레이션**: "팔을 뻗어도 닿지 않으면, 광선으로 가리켜 잡을 수 있습니다."
- **시각 힌트**: 컨트롤러에서 광선이 강조되어 표시. 큐브에 outline 글로우.
- **성공 조건**: Ray Interactor 로 큐브를 끌어와 박스 안에 1개 이상 Release.

### 5. 텔레포트
- **연출**: 사용자 위치에서 5m 떨어진 곳에 빛나는 원형 발판. 그 위에 다음 단계의 큐브.
- **내레이션**: "왼쪽 스틱을 앞으로 밀어 발판을 가리키고, 놓으면 이동합니다."
- **시각 힌트**: 컨트롤러 스틱에 화살표. 발판 위로 텔레포트 가이드 라인 미리 한 번 자동 시연.
- **성공 조건**: 사용자 위치가 TeleportTarget Trigger 안.
- **주의**: 멀미 방지를 위해 텔레포트 시 화면 페이드 (50ms 블랙).

### 6. UI 클릭
- **연출**: 텔레포트 후 정면에 월드 공간 UI 패널. "다음" 버튼 1개.
- **내레이션**: "이제 UI 버튼을 눌러볼게요. 광선을 버튼에 맞추고 트리거를 당기세요."
- **성공 조건**: `Button.onClick` 발화.
- **확장**: 추후 "이전/다음/일시정지" 3버튼 패턴으로 확장 가능.

### 7. 통합 챌린지
- **연출**: 다음 영역에 색깔별 슬롯 3개와 색깔 큐브 3개. 큐브와 슬롯이 서로 떨어진 위치.
- **내레이션**: "지금까지 배운 동작을 모두 사용해 큐브를 같은 색 슬롯에 끼워주세요."
- **성공 조건**: `XRSocketInteractor` 가 색상 매칭으로 3개 모두 Lock.
- **운영자 옵션**: 1개만 끼우면 통과시키는 "쉬움 모드" 토글 가능.

### 8. 완료
- **연출**: 모든 슬롯이 빛나며 축하 효과음 + 컨페티 파티클.
- **UI**: 큰 패널에 "본 게임 시작" / "다시하기" 두 버튼.
- **내레이션**: "준비 완료입니다! 즐거운 시간 되세요."
- **씬 전환**: `GameManager.LoadScene(SceneId.GameEntry)` 또는 `Tutorial` 재시작.

## 운영자 단축키 (개발/현장 운영용)
빌드 옵션으로 활성화. 일반 사용자 모드에서는 비활성.

| 입력 | 동작 |
|---|---|
| 양 그립 + 메뉴 버튼 동시 3초 | 운영자 메뉴 진입 |
| 운영자 메뉴 → "다음 단계" | 현재 단계 강제 통과 |
| 운영자 메뉴 → "리셋" | Mission_00 으로 점프 |
| 운영자 메뉴 → "본 게임" | 03_GameEntry 즉시 로드 |

## 사운드 / 자막 가이드
- 모든 내레이션은 **3~6초 이내**로 짧게.
- 자막은 한 줄당 ≤ 18자, 최대 2줄.
- 효과음: 성공 (밝은 차임), 실패 (부드러운 톤다운), 단계 진입 (짧은 띵).
